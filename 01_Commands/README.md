# Linux Commands

- Create a shorcut:
    - `ln -s /path/to/target ~/shortcut_name`

- Kill a process:
    - `kill -9 <PID>`

- To check all shared library dependencies of a program:
    - `ldd <file_name>` 

- To change the permissions of a file or directory:
    - `chmod +x <file_name>`
    - `chmod 777 <filename>`
